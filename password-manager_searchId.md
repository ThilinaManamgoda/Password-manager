## password-manager searchId

Search Password with ID

### Synopsis

You can use either complete or part of ID for searching

```
password-manager searchId [ID] [flags]
```

### Options

```
  -h, --help        help for searchId
  -s, --show-pass   Print password to STDOUT
```
